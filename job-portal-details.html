Project description

I developed a Job Portal web application that connects job seekers with employers through a streamlined and user-friendly platform. The system allows candidates to create profiles, upload resumes, and apply for jobs, while employers can post vacancies, manage applications, and filter candidates based on custom criteria. Built with Laravel, the portal features secure authentication, role-based access, and a responsive design to ensure accessibility across all devices.

Project Languages Used: HTML, CSS, Bootstrap, jQuery, PHP, Laravel, AJAX.
