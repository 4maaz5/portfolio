Project description

School Management System – A comprehensive SaaS platform designed specifically for Saudi educational institutions, built with PHP/Laravel and Flutter. The system digitizes core academic operations including student enrollment, attendance tracking, exam scheduling/grading, and automated fee management with integrated ZATCA-compliant e-invoicing to meet Saudi tax regulations. Features role-based portals for administrators, teachers, students, and parents, with real-time notifications and reporting. Developed companion iOS and Android mobile applications to extend accessibility, allowing parents to monitor attendance, grades, and fees while enabling teachers to manage classes remotely—creating a connected educational ecosystem across web and mobile platforms.

Project Languages Used: HTML, CSS, Bootstrap, jQuery, PHP, Laravel, AJAX.
